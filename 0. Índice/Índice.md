Esta es la guía básica para la partida de Dungeons and Dragons basada en Hollow Knight. Se expondrán aquí las [[Reglas básicas]] del juego contextualizadas y todo lo necesario para la creación de personaje, contextualizando las clases, subclases y razas para que tengan sentido dentro del mundo y siendo coherente con la historia.

También se incluirá el Lore del mundo y el contexto de cada región del juego. Solo se incluirá el conocimiento que tienen (o deberían) tener los personajes del juego. Adicionalmente, se harán resúmenes de cada sesión para poder recordarlo con mayor facilidad, descripciones de objetos consumibles, objetos mágicos, equipamiento y armas que consiga la party, descripciones breves de NPCS conocidos y misiones activas para un mejor seguimiento de toda la información.
